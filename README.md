# The Joy of CSS Grid
CSS Grid layout and animation examples

## Features
- Responsive 12-column bootstrap replacement
- Mosaic photo gallery
- Staggered animation grid

## Usage
1. Compile SCSS files \
`npm run build`

2. Open `public/index.html` in your browser.
